Overview:

ISS is a live software that uses open-notify.org's public API and keeps track of the International space station. By using the turtle graphics library, it displays the exact location of the station on the map. Some of the other info that you can get out of using this software are, the amount of people on this spaceship, as well as their names. It also predicts when it will be crossing over Indianapolis, IN.

Technologies Used:

  • Python
  • Turtle Graphics Library

Some of the lessons I got out of this project:

  • The use of requests package to query real world data
  • Deeper dive into dictionaries and indexing
  • Practical knowledge of the built-in Python Turtle Graphics Library
